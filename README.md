## How to understand server-side rendering?

### React client only
```
> cd react-client
> npm i
terminal_1 > npm run dev
terminal_2 > npm run mock
terminal_3 > curl http://localhost:8080
```

### React client + server-side
```
> cd react-server
> npm i
terminal_1 > npm run start
terminal_2 > npm run mock
terminal_3 > curl http://localhost:8080
```
